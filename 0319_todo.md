# 0319 上线待办清单

> 日期：2026-03-19
> 项目：虾虾工厂 / `factoryxiaxia`
> 目标：以“可正式对外销售和交付”为标准推进上线
> 首发范围：只正式上线 `Support Responder`；`WeChat Official Account Manager` 保留内测

---

## 一、上线范围定义

### 本次首发必须成立的业务闭环

- 用户可以搜索并购买官方数字员工
- 用户完成支付或商务确认后，可以创建部署实例
- 用户可以上传知识库、发布机器人、拿到公开 API
- 客户调用公开 API 可以拿到稳定中文回复
- 平台可以看到部署状态、会话、转人工、基础用量
- 出现异常时，平台可以定位、停用、恢复或回滚

### 本次不上线的范围

- 多 SKU 同时放量销售
- 微信公众号正式渠道直连
- 自动退款、自动改订单、自动执行高风险动作
- 复杂 CRM / 工单系统深度集成
- 私有化部署标准产品包

## 二、优先级说明

- `P0`：上线阻塞项，不做不能正式上线
- `P1`：上线前强烈建议完成，不做会影响交付质量
- `P2`：可在灰度后补，但需要提前排期

## 七、上线完成定义

当下面这些条件全部满足时，才算“可以正式上线”：

- 没有默认管理员、默认密钥、默认支付链路
- 公网只有受控入口，数据库和后端服务不裸露
- `public_token` 可轮换、可停用、有限流、有限额
- 已发布机器人走真实 Provider，而不是纯规则回复
- 部署、发布、公开 API、转人工、用量都有自动化回归
- 运维文档、客户接入文档、回滚手册都已齐备
- 至少完成一次灰度环境演练和一次回滚演练
